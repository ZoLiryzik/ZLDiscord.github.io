ZLDiscord v1.2

Minecraft ↔ Discord интеграция: привязка аккаунтов, голосования, эмбеды, кастомные команды с шаблонизатором

Введение

ZLDiscord — Paper-плагин, связывающий Minecraft-сервер с Discord-каналом. Поддерживает Paper 1.16.5+ (Java 11+).

  • Привязка Minecraft аккаунтов к Discord
  • Голосования с таймером через Discord Embed
  • Отправка и управление Embed-сообщениями из Minecraft
  • Кастомные команды с мощным шаблонизатором
  • PlaceholderAPI — %player_name%, %server_online%
  • Прямой запуск команд: /hello в чате, z!hello в Discord

Установка

  1. Скачать ZoLiryzik.jar (14MB, JDA встроен)
  2. Положить в plugins/
  3. Запустить сервер — создастся plugins/ZLDiscord/config.yml
  4. Настроить bot token, guild ID, role IDs
  5. Перезапустить сервер
Важно: Требуется Java 11+. Discord Bot Token создаётся в Discord Developer Portal. Включите Intents: GUILD_MESSAGES, DIRECT_MESSAGES, MESSAGE_CONTENT, GUILD_MEMBERS, GUILD_PRESENCES, GUILD_VOICE_STATES. 
# Зависимости (soft):
- PlaceholderAPI (опционально, но рекомендуется)

Привязка аккаунтов

  1. Войти на сервер, написать /zlink
  2. Получить 6-значный код (действует 5 минут)
  3. В Discord написать в ЛС боту /link код
  4. Готово. Привязанный Discord-ID сохраняется в links.yml

Плейсхолдер %zoliryzik_in_vc% показывает, находится ли привязанный пользователь в голосовом канале.

Кастомные команды

Создание

  • Minecraft: /zcc create hello "Привет, %player_name%!"
  • Discord: /cc create — откроет модальное окно
  • Файл: Править custom_commands.yml, затем /zcc reload

Выполнение

ОткудаСинтаксисПример
Minecraft чат/имя_команды [args]/hello ZoLiryzik
Discord текстz!имя_команды [args]z!hello ZoLiryzik
Discord slash/cc run имя [args]/cc run hello ZoLiryzik
Minecraft/zcc run имя [args]/zcc run hello ZoLiryzik

Формат custom_commands.yml

commands:
  hello:
    template: "<~ papi(%player_name%) ~>, привет!"
    description: "Приветствие"
    cooldown: 10          # секунд между использованиями
    aliases: ["hi"]
    enabled: true

  heal:
    template: |
      <do mc.command("heal " + args[0])>
      <~ args[0] ~> вылечен!
    description: "Вылечить игрока"
    cooldown: 30
    aliases: []
    enabled: true

PlaceholderAPI

Шаблоны автоматически обрабатывают %placeholder% через PAPI. Доступны все установленные плейсхолдеры.

# Прямо в тексте (автоматически):
Привет, %player_name%! Онлайн: %server_online%

# Через функцию papi():
<~ papi(%player_name%) ~>
<~ papi(%level%, args[0]) ~>     <!-- Для конкретного игрока -->
<~ papi(%server_online%) ~>

Собственный плейсхолдер: %zoliryzik_in_vc%

Показывает, находится ли привязанный Discord-пользователь в голосовом канале. Кэшируется на 60 секунд.

Bridge-действия

Bridge — действия, которые выполняются при рендеринге шаблона, но не выводят текст.

Minecraft Bridge

ДействиеОписание
<do mc.command("команда")>Выполнить консольную команду
<do mc.broadcast("текст")>Broadcast с цветами (&)
<do mc.give("ник предмет количество")>Выдать предмет через /give

Discord Bridge

ДействиеОписание
<do discord.send("channelId", "текст")>Отправить сообщение в канал
<do discord.embed("channelId", ...)>Отправить embed (title, description, color, image, fields)
<do discord.reaction("✅")>Добавить реакцию к последнему embed
<do discord.button("style", "id", "label")>Кнопка под последним embed

Пример шаблона

<set target = args[0]>
<if target == null>
  <return "Укажи ник!">
</if>
<do mc.command("heal " + target); mc.broadcast("&a" + target + " вылечен!")>
<do discord.send("123456789", target + " вылечен!")>
Игрок <~ target ~> вылечен. Здоровье: <~ papi(%player_health%, target) ~>

Частые проблемы

ПроблемаРешение
Бот не отвечает в DiscordПроверить токен и Intents в Discord Developer Portal. Проверить main-discord-guild-id
/команда не работает в MinecraftВыполнить /zcc reload. Убедиться что команда включена (enabled: true)
z!команда не работает в DiscordБот должен иметь доступ к каналу. Убедиться что команда существует через /cc list
PlaceholderAPI не работаетУстановить PlaceholderAPI в plugins/. Плагин пишет лог при старте
%player_name% не заменяется в DiscordВ Discord PAPI работает только с papi(%placeholder%) функцией, не автоматом
Ошибка "SimplePluginManager deprecated"Не влияет на работу. Плагин работает на всех Paper 1.16.5+
JDA не запускаетсяПроверить интернет-соединение сервера. JDA требует доступ к Discord API
Файлы: Все данные хранятся в plugins/ZLDiscord/: config.yml, custom_commands.yml, embeds.yml, links.yml, lang_ru.yml, lang_en.yml
