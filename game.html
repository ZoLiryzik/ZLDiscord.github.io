🔫 Оружие Без ограничений

  • Спрайт / звук выстрела
  • Скорострельность / магазин / перезарядка
  • Тип ствола (залп, веер, кольцо, змейка…)
  • Снаряд (пуля, луч, крюк, волна, мина…)
  • Траектория (прямо, дуга, синус, спираль, бумеранг…)
  • Скорость / время жизни / пробивание
  • Отскок / прилипание / самонавод
  • Разделение в полёте
  • Цепь между снарядами
  • Триггеры: при_выстреле / при_попадании / при_убийстве / при_промахе / при_перезарядке / при_отскоке / каждые_N_секунд
  • Действия: спавн / урон / телепорт / щит / лечение / невидимость / замедление / притяжение / обмен / взрыв / стена / кража / отражение / полёт / призыв
  • Хост придумывает свои действия → кладёт .txt скрипт в папку

👾 Враг / Босс Без ограничений

  • Спрайт / звуки
  • Здоровье / скорость / хитбокс
  • ИИ: преследование, патруль, засада, убегание, хаос
  • Ближняя атака (таран, захват, отскок)
  • Дальняя атака (свой снаряд, кд, тип)
  • Элитность (аура, множитель хп)
  • Призыв подкрепления при хп%
  • Дроп (предмет, шанс, количество)
  • Реакция на попадание
  • Реакция на смерть (взрыв, разделение, облако)
  • Босс: фазы, ультимейт, смена поведения, арена
  • Любые кастомные действия хоста

  • Тайлсет (пол, стены, двери)

  • Свои правила через .txt скрипт

🗺️ Карта (забег) Хост настраивает

📝 Формат .txt Простота

Одна строка = один параметр. Хост пишет что хочет:

ключ: значение
триггер: действие параметр=число
мой_выдуманный_ключ: да

Неизвестные движку ключи → поиск .txt скрипта с таким именем в папке.
Нет скрипта → запись в лог, игра не падает.

Движок — исполнитель. Фантазия хоста — контент. Ограничений нет.
