Вывод по сравнению

ZLDiscord — не очередной чат-бридж. В отличие от DiscordSRV, NexDiscordLink, Synccord и других Minecraft-плагинов, ZLDiscord не ограничивается пересылкой сообщений между Minecraft и Discord. Его ключевое отличие — возможность управлять Discord-сервером прямо из Minecraft через мощный шаблонизатор, создавать embed-сообщения, голосования, модальные окна, реакции и меню выбора ролей — всё через YAML-конфиги без единой строчки кода.

HTTP LongPoll и DataStore — функции, которых нет ни в одном другом плагине. LongPoll позволяет внешним веб-скриптам взаимодействовать с сервером Minecraft, а DataStore — хранить произвольные атрибуты бота, участников Discord и игроков, доступные в шаблонах. Cross-Server Command Queue решает проблему синхронизации команд на сети серверов.

Единственное пересечение с аналогами — стандартные функции (чат-бридж, привязка аккаунтов, синхронизация ролей), которые есть в любом Discord-Minecraft плагине. Но даже в этих базовых вещах ZLDiscord идёт дальше: кастомные команды работают одновременно в Minecraft и Discord, а языковые файлы поддерживают полную локализацию.

Итог: ZLDiscord — это гибрид Minecraft-плагина и Discord-бота с уникальным набором функций, не имеющий прямых аналогов. Плагиат исключён.

ЗАКЛЮЧЕНИЕ: ПЛАГИАТА НЕТ

ZLDiscord не является плагиатом ни одного из существующих Discord-Minecraft плагинов или Discord-ботов. Плагин содержит 8 уникальных функций, отсутствующих во всех аналогах вместе взятых.

Уникальные функции ZLDiscord

#ФункцияОписаниеАналоги
1 Шаблонизатор <~ ~> Мощный генератор embed-сообщений с функциями-чейнами: guild.channel.setName, embed.addBlankField, member.setNickname. Поддерживает криптографию и HTTP-запросы. —
2 HTTP LongPoll Позволяет внешним скриптам (PHP, Python, JS) отправлять запросы в Minecraft-сервер. Команды LongPoll: создание, листинг, удаление, перезагрузка. —
3 Embed CRUD из Minecraft Полное создание, редактирование, удаление Discord embed-сообщений прямо из Minecraft команд /zembed и /zembedmanage. —
4 Голосования с таймером Команда /zevent с таймером автоматического завершения. Настраиваемые варианты голосования, результирующая MC-команда. —
5 DataStore (атрибуты) Хранилище bot/member/player атрибутов, доступных в шаблонах. Команды get/set/inc/dec/list через /zattr. —
6 Модалки + SelectMenu Discord Modal и SelectMenu (Dropdown) UI-компоненты, привязанные к Minecraft-логике. Настраиваются через modals.yml. —
7 Кастомные команды с двойным scope Команды работают и в Minecraft (Bukkit) и в Discord (slash). Синтаксис <~ ~> в ответах, модалки, роли, embed, атрибуты, chain-обработчики. —
8 Cross-Server Command Queue Очередь команд, выполняющихся на одном сервере сети (BungeeCord-подобной). Синхронизация через MySQL. —
